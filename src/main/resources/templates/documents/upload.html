Supported file types: PDF, DOCX, DOC, TXT. Maximum file size: 10MB.
What happens after upload?
  1. Your document will be uploaded and stored securely.
  2. Text will be extracted from the document.
  3. AI analysis will be performed to generate:
    • A concise summary of the document
    • Key keywords and phrases
    • Sentiment analysis
    • Document categorization
  4. You'll be redirected to the documents list where you can view the processing status.
